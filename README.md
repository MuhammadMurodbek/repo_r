# React-DND-Scratch

> This is sample repository demonstrates how to use React and Drag and Drop without any third-party library

## Starting the development server

```shell
npm start
```

## Building the `bundle`

```shell
npm run build
```

## Type-Checking

```shell
npm run type-check
```

And to run in --watch mode:

```shell
npm run type-check:watch
```
